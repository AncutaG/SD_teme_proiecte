# SD_teme_proiecte
Tema 1 pentru Structuri de Date Unibuc.

Cuprinde o analiza succinta a timpilor de rulare (in cadrul diferitelor teste) pentru 5 dintre algoritmii de sortare:
- Radix sort (a fost urmarit pe doua baze 10 si 16)
- Merge sort
- Shell sort
- Quick sort
- Counting sort
